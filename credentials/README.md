Place your Google Cloud service account JSON key file here.
Rename it to: service_account.json

How to get one:

1. Go to <https://console.cloud.google.com/>
2. Create a project (or use existing)
3. Enable "Google Sheets API" and "Google Drive API"
4. Go to IAM & Admin → Service Accounts → Create Service Account
5. Create a key (JSON format) → download
6. Place the file here as service_account.json
7. Share your Google Sheet with the service account email (the one ending in @...iam.gserviceaccount.com)
